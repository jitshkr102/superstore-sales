# superstore-sales
# Superstore Sales Analysis 📊

## Overview
This project is an Exploratory Data Analysis (EDA) on Superstore sales data using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Analysis
- Region-wise sales analysis
- Category and sub-category analysis
- Monthly and yearly sales trends
- Segment-wise performance
- Pivot table insights

## Outcome
Identified key revenue drivers and sales patterns across different dimensions.
